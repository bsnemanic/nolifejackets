# README for No Life Jackets

## Overview

Welcome to the No Life Jackets project! This is a simple Jekyll website designed to showcase podcast episodes and provide a platform for blogging. 

## Features

- **Podcast Episodes**: Links to episodes available on YouTube and Spotify.
- **Blog Section**: A space to share thoughts, updates, and articles.

## Project Structure

- `_config.yml`: Configuration settings for the Jekyll site.
- `CNAME`: Custom domain name for the website.
- `Gemfile`: Lists Ruby gems required for the Jekyll site.
- `index.md`: Main landing page with links to podcasts and blog posts.
- `about.md`: Information about the website and its creator.
- `podcast/index.md`: Main page for the podcast section.
- `_posts/`: Directory containing blog posts.
- `_data/podcasts.yml`: Structured data for podcast episodes.
- `_layouts/`: Layout templates for different sections of the site.
- `_includes/`: Reusable HTML components like header and footer.
- `assets/`: Contains CSS and JavaScript files for styling and interactivity.
- `.gitignore`: Specifies files to be ignored by Git.

## Getting Started

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd nolifejackets
   ```

3. **Install dependencies**:
   ```
   bundle install
   ```

4. **Serve the site locally**:
   ```
   bundle exec jekyll serve
   ```

5. **Open your browser** and go to `http://localhost:4000` to view the site.
